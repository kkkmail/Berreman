## Reconciler feedback applied (round >= 2)

- **F-001 (blocker, meaning-drift) — APPLIED.** Slice 001 owns Part A but
  had replaced the parent's eight Part-A acceptance criteria
  (`AC-A1..AC-A8`, parent `.spec-md:772-779`) with a *different*,
  directive-derived list carrying the same `AC-A*` identifiers, so the
  same token denoted two different claims across the slice set (e.g.
  parent `AC-A2` = schema-validation-on-load vs. slice `AC-A2` = engine
  reuse; parent `AC-A5` = SI-storage vs. slice `AC-A5` = UI architecture;
  parent `AC-A7` = eV/cm⁻¹ round-trip vs. slice `AC-A7` = numerics
  boundary). Sibling slices (002, 003, 009) cite the parent meanings under
  those labels, so this was a live collision. Fixed by reproducing the
  parent's `AC-A1..AC-A8` **verbatim** under "Acceptance criteria owned by
  this slice", matching the format slices 002–010 already use.

- **F-002 (blocker, invented) — APPLIED.** Slice 001 carried a net-new
  `AC-A9` (audit gate) with no parent counterpart (the parent's Part-A
  acceptance list ends at `AC-A8`). Removed. The audit-gate substance is
  already expressed by the parent's verbatim `AC-A8`, which is now present.

- **F-003 (info, invented) — REVIEWED, RETAINED.** Slice 001 lists
  `OpticalConstructor.Domain/OpticalConstructorProject.fs` (new), a file
  not named in the parent's Files-in-scope list. The judge recorded this
  as **info only**, expressly traceable to §A.7 ("Define the root
  aggregate `OpticalConstructorProject`") and a reasonable realization
  rather than a manufactured artefact, flagged for the operator's audit
  trail. No contractful drop or invention; retained as-is.

- **Consistency follow-through.** Slice 001's `Testing plan` carried loose
  parenthetical AC references keyed to the now-removed drifted meanings
  (e.g. "JSON round-trip … (AC-A6)" where parent `AC-A6` is the `.binz`
  sidecar rule). Updated those non-contractful pointers to match the
  verbatim parent meanings (`AC-A1`, `AC-A2`, `AC-A3`, `AC-A4`) so the
  worker guidance and the acceptance block agree.

No findings were judged stale or refuted; all substantive findings were
applied.
